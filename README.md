# Smart Kiosk System

A futuristic Smart Food Ordering Kiosk System built using:

- HTML
- CSS
- JavaScript
- Java Servlets
- MySQL

## Features

- Login & Register
- Smart Dashboard
- Food Menu
- Cart System
- Billing & Payment
- Payment History
- Google Maps Integration
- Theme Toggle
- Live Clock
- AI Assistant UI

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Java Servlet
- Database: MySQL
